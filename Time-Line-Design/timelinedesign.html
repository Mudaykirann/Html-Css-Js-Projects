HTML

Hyper Text Markup Language

The standard language used to structure and present content on the web, defining the elements and layout of a web page.

CSS

Cascading Style Sheets

A style sheet language that controls the visual appearance of HTML elements, allowing for customization of colors, fonts, spacing, and more.

Java Script

Java Script God Of Programming Language

A versatile programming language that enables interactivity and dynamic behavior on web pages, allowing for client-side scripting and manipulation of HTML and CSS.

Bootstrap

CSS Framework

Popular CSS framework that provides pre-designed components, responsive grid layouts, and styling options to facilitate faster and more consistent web development.

ReactJs

JS Framework

JavaScript library for building user interfaces, offering a component-based approach that enables efficient, reusable, and scalable web application development.

Git

Version Control Systems

Distributed version control system that tracks changes to files, facilitates collaboration among developers, and allows for efficient management of software projects by keeping a history of changes and enabling easy branching and merging.
